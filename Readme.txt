Данный программный модуль тарифицирует абонента с IP-адресом 192.168.250.27 с коэффициентом k:
1руб/Mб. Но так как абонент потратил меньше 1 Мб, было принято решение 
протарифицировать абонента с коэффициентом 1руб/Кб.

Текст программы содержится в файлах Program.cs, Charts.cs, Data.cs,
Histogram.cs, HistogramSample.cs.
Файл с исходными данными: bin/Debug/data.csv
Запуск программы производится с помощью exe-файла bin/Debug/Names.exe
На консольный экран выводится количество потраченных битов,
а также сумма в рубляхпри разных тарифах.
В оотдельном окне, выводится график количества трафика в зависимости от времени.
График можно увеличить или уменьшить при помощи колесика мышки. 
Скриншот вывода представлен в файлах output1.png и output2.png

Разработка программного модуля производилась в Microsoft Visual Studio на языке С#
Для запуска необходим .Net Framework v4.8 
Также в папке находится отчет и титульный лист в файлах с соответствующими названиями
